# List Age Groups
This API will return a list of travel type.  


## URL
https://\<your base URL\>/api

## Header parameters
**function**: 3

## Results 
```
{
  "result": bool, 
  "data": [
    {        
        "id": integer,            
        "description": string          
    }
  ]
}
```

The travel type id is required when searching for tours.
